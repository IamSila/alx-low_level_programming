# variadic functions in C
- These are are functions which may take a variable number of arguments and are declared with an ellipsis in place of the last parameter.
## stdarg.h
- argument that allows functions in C to take indefinate number of arguments...
- va_list - type for iterating thru the arguments
- va_start - start iterating a va_list
- va_end - free a va_list
- va_copy - copyt the contents of a va-list
- va_arg - retrieve an argument
